Cobo 支付解决方案目前支持两种主流稳定币：**USDC** 和 **USDT**，并可在多个主流公链上使用。本文档详细说明各币种和公链组合在收款和 Off-Ramp 对接场景下的支持情况。

## 收款支持的币种和公链

Cobo 支付解决方案支持以下币种和公链组合进行收款：

| 稳定币 | 支持的公链 | Token ID |
|--------|------------|----------|
| **USDC** | Ethereum | ETH_USDC |
| | Arbitrum | ARBITRUM_USDCOIN |
| | Base | BASE_USDC |
| | Polygon | MATIC_USDC2 |
| | BNB Smart Chain | BSC_USDC |
| | Solana | SOL_USDC |
| **USDT** | TRON | TRON_USDT |
| | Ethereum | ETH_USDT |
| | Arbitrum | ARBITRUM_USDT |
| | Base | BASE_USDT |
| | Polygon | MATIC_USDT |
| | BNB Smart Chain | BSC_USDT |
| | Solana | SOL_USDT |


同时，您也可以使用 TTRON (TRON 测试链) 在开发环境进行测试。Cobo 支付解决方案目前支持测试币 TTRON_USDT1：

* 代币合约地址：`TXYZopYRdj2D9XRtbG411XZZ3kM5VkAeBf`
* 测试币水龙头：https://nileex.io/join/getJoinPage


## Off-Ramp 对接支持情况

以下表格列出了各币种和公链组合在不同环境中的 Off-Ramp 对接支持情况：

| 币种和公链 | 开发环境 | 生产环境 |
|---------|----------|----------|
| ETH_USDC | ✅ 支持 | ✅ 支持 |
| ARBITRUM_USDCOIN | ❌ 不支持 | ✅ 支持（需跨链费用）|
| BASE_USDC | ❌ 不支持 | ✅ 支持（需跨链费用）|
| MATIC_USDC2 | ❌ 不支持 | ✅ 支持（需跨链费用）|
| BSC_USDC | ❌ 不支持 | ✅ 支持（需跨链费用）|
| TRON_USDT | ✅ 支持 | ✅ 支持 |
| ETH_USDT | ✅ 支持 | ✅ 支持 |
| ARBITRUM_USDT | ❌ 不支持 | ✅ 支持（需跨链费用）|
| SOL_USDC | ❌ 不支持 | ✅ 支持（需跨链费用）|
| SOL_USDT | ❌ 不支持 | ✅ 支持（需跨链费用）|
| BASE_USDT | ❌ 不支持 | ❌ 不支持 |
| MATIC_USDT | ✅ 支持（需跨链费用）| ✅ 支持（需跨链费用）|
| BSC_USDT | ✅ 支持（需跨链费用）| ✅ 支持（需跨链费用）|
| TTRON_USDT | ✅ 支持 | ❌ 不支持 |

<Info>标记为"需跨链费用"的币种和公链组合在进行 Off-Ramp 对接时，除常规费用外，还需额外支付跨链费用。</Info>
