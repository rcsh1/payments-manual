## 概述

本文档帮助您了解不同支付场景下支持的币种、公链和货币：

- **收款**：接收客户的稳定币付款
- **定价**：使用熟悉的法币设置价格
- **Off-Ramp 对接**：将稳定币兑换为法币

<Note>
本文档列出的代币是 Cobo Payment 支持的币种。虽然 Cobo 托管平台支持更广泛的代币种类，但只有下方列出的稳定币在 Payment 收款时才会自动入账。发送到 Payment 地址的其他代币将不会被自动处理。
</Note>

### 如何选择合适的公链

| 优先级 | 推荐公链 | 原因 |
|--------|----------|------|
| 最低手续费 | BNB Smart Chain、Arbitrum、Base | Layer 2 或低成本网络 |
| 最广泛兼容性 | Ethereum | 支持范围最广 |
| 可直接 Off-Ramp（无跨链费用） | Ethereum (ETH_USDC, ETH_USDT)、TRON (TRON_USDT) | 无需支付额外跨链费用 |

## 重要说明

<Warning>
**发送前务必确认网络**

在错误的网络上发送代币可能导致资金永久丢失。例如：
- 将 Polygon 网络上的 USDC 发送到 Ethereum 地址将无法入账
- 每个 Token ID 对应特定的公链 — 请务必准确匹配

如果您或您的客户不慎将代币发送到不支持的网络，请携带交易哈希联系 [Cobo 支持](https://support.cobo.com) 寻求帮助。
</Warning>

<Note>
**Token ID 使用说明**

在与 Cobo API 集成时，请务必使用本文档中指定的准确 Token ID（如 `ETH_USDC` 或 `TRON_USDT`）。使用错误的 Token ID 将导致 API 报错。
</Note>
